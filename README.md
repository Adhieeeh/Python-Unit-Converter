# Simple Unit Converter

A lightweight, interactive Command Line Interface (CLI) tool built with Python to perform common unit conversions. This project focuses on clean, readable code and demonstrates fundamental programming logic.

##  Features
- **Temperature:** Celsius to Fahrenheit.
- **Distance:** Kilometers to Miles.
- **Weight:** Kilograms to Pounds.
- **Interactive Menu:** Easy-to-use terminal interface for user selection.
- **Modular Code:** Logic is broken down into reusable functions.

##  Concepts Applied
This project allowed me to practice several core Python skills:
- Defining and calling **functions**.
- Handling **User Input** through the terminal.
- **Data Type Conversion** (converting strings to floats for math).
- **Conditional Logic** (if-elif-else statements).
- String formatting for clean output.
